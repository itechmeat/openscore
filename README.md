# OpenScore

## Project setup
```
npm install
```

## Firebase setup
```
npm install -g firebase-tools
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Deploy to Firebase
```
firebase deploy
```

### Font for digits
